# Diego Hartwig — Portfolio

Single Page Application desenvolvida para o site  `diegohartwig.com.br`, com foco em apresentar minha trajetória, especialidades e atuação nas áreas de Engenharia de Dados, Inteligência Artificial Aplicada e Engenharia de Software.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
